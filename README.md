# Petshop Online - Projeto Acadêmico

Uma plataforma web completa para um petshop online, desenvolvida com **HTML, CSS, JavaScript e PHP**.

## 📋 Características

### Fase 1 - Fundamentação
- Página inicial (Home) com apresentação
- Estrutura HTML sem CSS/Bootstrap
- Header com navegação
- Body com 2 produtos de cada 3 categorias
- Serviços destacados
- Footer

### Fase 2 - Implementação Completa
- CSS/Bootstrap responsivo
- JavaScript para interatividade
- Carrossel de produtos
- Formulário de cadastro (cliente + pet)
- Agendamento com calendário
- Tele-busca e entrega
- Acessibilidade
- Comentários de clientes
- Estrutura profissional

## 🚀 Como Usar

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/petshop-php.git
cd petshop-php
```

2. Configure um servidor PHP local (XAMPP, WAMP, LAMP):
```bash
# Se usar PHP built-in
php -S localhost:8000
```

3. Acesse no navegador:
```
http://localhost:8000
```
